# 小红蚁 (Little Red Ant) - 产品用户手册与功能概要

## 1. 产品概述
本产品是一款专为小红书创作者设计的辅助工具，深度融合了 **AI 内容生成** 与 **RPA (机器人流程自动化)** 技术。旨在帮助用户快速生成高质量的种草文案，并通过自动化脚本完成繁琐的发布流程，显著提升内容创作与运营效率。

## 2. 核心功能

### 2.1 AI 内容生成
利用先进的大语言模型，根据用户输入的关键词或主题，自动生成符合小红书风格的：
- **吸引人的标题**：包含Emoji和爆款关键词。
- **正文内容**：采用“种草”风格，分段清晰，语气亲切。
- **标签 (Tags)**：自动匹配高热度话题标签。

### 2.2 RPA 一键发布
通过模拟人工操作，自动将生成的内容填充至小红书发布页面。核心特点是“一键启动，人机协作”。

## 3. 核心流程与使用指南

本章节详细说明如何使用“一键发布”功能，特别是需要用户介入的**手动上传图片**环节。

### 3.1 操作流程图
```mermaid
graph TD
  A[输入主题/关键词] --> B[点击生成内容]
  B --> C{检查生成结果}
  C -->|满意| D[点击"一键发布"]
  D --> E[浏览器自动打开/跳转发布页]
  E --> F[**RPA自动填充标题、正文、标签**]
  F --> G[**用户手动上传图片**]
  G --> H[用户检查并点击发布]
```

### 3.2 详细步骤说明

1.  **生成内容**：
    在软件界面输入想要创作的主题，点击“生成”按钮。等待AI生成标题、正文和标签。

2.  **启动发布**：
    确认内容无误后，点击界面上的 **“一键发布”** 按钮。

3.  **RPA 自动填充**：
    - RPA 脚本会自动启动浏览器并打开小红书创作者中心发布页面。
    - **系统自动操作**：程序会自动将生成的标题、正文和标签填写到对应的输入框中。

4.  **手动上传图片 (用户介入)**：
    - 在文字内容填充完毕后（或者同时），**您需要点击页面上的“上传图片”区域**。
    - 在弹出的系统文件选择窗口中，**手动选中您要发布的图片并上传**。
    - *注意*：由于浏览器安全限制，RPA 脚本目前仅负责文本填充，图片上传需人工完成。

5.  **最终发布**：
    - 检查图片上传成功，且标题正文无误后。
    - **手动点击页面底部的“发布”按钮**，完成笔记发布。

## 4. 用户界面设计概要
- **风格**：简洁、直观的桌面应用风格。
- **布局**：
    - **内容生成区**：包含主题输入框和生成按钮。
    - **预览编辑区**：显示生成的标题、正文和标签，支持用户手动修改。
    - **操作区**：醒目的“一键发布”按钮，触发 RPA 流程。
