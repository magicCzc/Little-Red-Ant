# 📕 小红书推广笔记文案包 (防限流版)

## ⚠️ 发布前必读：为什么原版文案不能直接发？

如果在小红书直接发之前的 Markdown 版本，大概率会遇到以下问题：
1.  **被限流/判违规**：小红书对**“脚本”、“爬虫”、“RPA”、“自动化软件”、“防封号”、“引流”**等词汇极其敏感，系统会判定为“营销推广”甚至“违规外挂”，导致笔记没人看，甚至账号被关小黑屋。
2.  **用户看不懂/不爱看**：小红书用户喜欢**真诚分享、种草、实用干货**的风格。太硬的广告（“我是xxx软件”）和太技术的词（“API”、“多模态”）会让用户直接划走。
3.  **排版乱**：小红书不支持 Markdown 语法（# 标题），需要用 Emoji 和空行来排版。

## 🛡️ 防坑小贴士

1.  **不要放链接**：正文和评论区都**绝对不要**放下载链接或网址，必被屏蔽。可以说“主页有惊喜”或“私信”。
2.  **敏感词替换**：
    *   ❌ 脚本/外挂/RPA -> ✅ 自动化/黑科技/小助手/小工具
    *   ❌ 爬虫/抓取 -> ✅ 搜集灵感/看热点
    *   ❌ 引流/加V -> ✅ 涨粉/获客
3.  **引导互动**：结尾一定要引导用户评论（“求分享”、“蹲”），评论多了笔记权重才会高。
