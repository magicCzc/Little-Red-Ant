# 小红书智能起号工具 V2.0 产品需求文档（PRD）

## 1. 项目概述
### 1.1 项目背景
针对零基础个人创作者（宝妈、上班族等）面临的“选题难、创作难、运营难”痛点，开发一款整合“热点发现-智能创作-合规发布”的自动化工具。
**V2.0 更新**: 针对短视频赛道，新增“视频创作工作台”，解决视频脚本创作门槛高、逻辑混乱的问题。

### 1.2 产品目标
*   **MVP目标**: 帮助用户48小时内产出并发布第一篇高质量笔记。
*   **V2.0目标**: 实现“爆款视频结构”到“新视频脚本”的无缝迁移（Remix），提供所见即所得的分镜创作体验。

## 2. 核心用户画像 (Persona)
*   **典型用户**: 林小微，28岁在职宝妈。
*   **场景**: 利用碎片时间（午休、睡前）快速生成并发布内容。
*   **需求**: 亲子/生活类内容，操作简单，手机端适配。

## 3. 功能需求详情

### 3.1 人设初始化问卷 (Priority: P0)
*   **强制引导**: 首次使用必填。
*   **字段**:
    *   **专注领域**: 美妆、母婴、家居等（单选）。
    *   **身份标签**: 如“95后新手妈妈”（支持自定义）。
    *   **账号风格**: 干货、治愈、搞笑等（带示例）。
    *   **对标账号**: 输入1-3个主页链接。
*   **输出**: 生成 User Profile，用于AI Prompt上下文。

### 3.2 智能创作工作台 (Core) - V2.0 重构

本模块通过**场景隔离**策略，将图文与视频创作彻底分流。

#### 3.2.1 图文笔记模式 (Note Mode)
*   **驱动逻辑**: **热点驱动**。
*   **适用场景**: 追热点、发图文、写长文。
*   **输入**: 
    *   每日热点榜单（微博/小红书/抖音）。
    *   创作类型选择：`图文笔记` 或 `深度长文`。
*   **功能**:
    *   **热点灵感**: 接入全网热榜，点击即可填入选题。
    *   **AI 配图**: 自动根据文案生成 3:4 静态配图（多图轮播）。
    *   **文案生成**: 标题（爆款公式）、正文（干货/体验/讨论）、标签。

#### 3.2.2 视频脚本模式 (Video Script Mode) - **V2.0 New**
*   **驱动逻辑**: **结构驱动** (Structure-Driven)。
*   **适用场景**: 仿写爆款视频、创作口播脚本。
*   **核心改进**:
    *   **场景纯净**: 自动**隐藏**“今日热点”和“AI静态配图”模块，避免干扰和 Token 浪费。
    *   **结构可视化**:
        *   **仿写源卡片**: 若从爆款库进入，显示原视频的“视觉风格提取”和“结构骨架”。
        *   **黄金法则**: 若直接创作，显示“黄金前3秒”等短视频创作指南。
*   **输入**:
    *   **新选题**: 用户只需输入新主题，AI 自动进行“结构迁移”（保留骨架，替换血肉）。
    *   **视觉关键词**: 引导输入如“快节奏、特写、赛博朋克”等指令。
*   **输出 (Storyboard)**:
    *   **分镜卡片**: 替代传统的 Markdown 表格，以卡片形式展示每一镜。
    *   **视觉指令**: 包含具体的运镜（推拉摇移）、灯光、道具描述。
    *   **闭环生成**: 每个分镜卡片内置**“生成此镜头视频”**按钮，一键调用文生视频模型。

### 3.3 安全发布策略 (Priority: P0)
*   **方式**: 模拟操作 (RPA)。
*   **功能**:
    *   **一键发布**: 立即发布（带随机延迟 ±30分钟）。
    *   **定时发布**: 指定时间。
    *   **风险提示**: 强制弹窗确认内容合规。

### 3.4 热点与对标 (Priority: P1)
*   **热点推送**: 每日更新垂类话题。
*   **对标监控**:
    *   **深度拆解**: 支持视频/图文的深度分析（视觉、结构、钩子）。
    *   **结构提取**: 将爆款内容转化为可复用的 JSON 结构模板。

## 4. 非功能性需求
*   **平台**: 移动端优先的 Web (PWA)。
*   **响应速度**: AI生成 ≤ 15秒。
*   **安全性**: 账号Cookies本地加密存储。

## 5. 技术决策
*   **前端**: React + Tailwind (Mobile-first).
*   **发布**: RPA (Playwright/Selenium) 模拟。
*   **AI**:
    *   Text: DeepSeek / Qwen-Max.
    *   Vision: Qwen-VL (视频/图片理解).
    *   Video Gen: Wan2.6 / Kling (需接入).
